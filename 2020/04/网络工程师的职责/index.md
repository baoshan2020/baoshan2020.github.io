# 网络工程师的职责






在大型企业中，网络工程师的工作职责可能还包括要监管一个或多个位置的网络。



此外，网络工程师可能还要帮助开发新的网络系统，以帮助推动公司或组织的发展。这种网络系统可包括本地网络、广域网、存储区域网络、虚拟LAN以及本地内部网或外部网服务器及资源。



网络工程师的工作内容和工作职责各有不同，一定程度上取决于企业是否依赖于托管服务提供商。



**但总体来说，职责包括以下：**



1. 管理和维护网络，以及网络性能监控
2. 发现、安装和维护网络更新
3. 故障排除和修复
4. 监管用户网络访问
5. 制定备份/灾难恢复计划以应对网络紧急事件
6. 为新应用和服务规划网络基础设施
7. 培训网络人员来为最终用户提供支持
8. 采购和维护网络设施及基础设施
9. 整合移动设备到企业的网络，无论是企业还是个人设备（BYOD）



网络工程师的职责还包括定期与IT团队成员开会，了解一个或多个业务部门的IT需求，无论是从运营、利润还是服务角度。但是并不需要经常与最终用户直接联系，但他们可培训支持人员这样做。



通常网络工程师需要计算机学士学位，多年网络系统工程师的工作经验。根据企业的情况，管理员/网络管理员认证会有所帮助，常见网络工程师认证包括思科、华为、Juniper Networks、VMware等供应商的认证。
