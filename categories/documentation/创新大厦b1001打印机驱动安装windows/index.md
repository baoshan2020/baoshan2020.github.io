# 创新大厦B1001打印机驱动安装(windows)


打印机使用说明书

下载驱动程序

第一步：右键点击开始-\>运行或win+R键

![](http://it.tusenergy.com/wordpress/wp-content/uploads/2020/03/%E5%9B%BE%E7%89%87.png)



第二步：点击运行，输入 [\\\\192.168.2.99](file:///\\192.168.2.992)

![](http://it.tusenergy.com/wordpress/wp-content/uploads/2020/03/图片-1.png)

用户名：guest 密码：空 并选择记住密码

![](http://it.tusenergy.com/wordpress/media/print-00.png)

第三步：进入安装界面后

![](6bc887701797e458f582cd208593bfab.png)

双击安装

![](http://it.tusenergy.com/wordpress/wp-content/uploads/2020/03/%E5%9B%BE%E7%89%87-2.png)

1.  安装打印机

第一步：选择——打印机-\>“设置默认打印机

![](http://it.tusenergy.com/wordpress/wp-content/uploads/2020/03/%E5%9B%BE%E7%89%87-3.png)

第二步：选择——打印机-\>“属性”

![](http://it.tusenergy.com/wordpress/wp-content/uploads/2020/03/%E5%9B%BE%E7%89%87-4.png)

第三步：配置-\>用户信息的设定-\>用户识别符选项中选择：**输入所有者名称**，用户识别符为：本人小写姓名全拼；

![](http://it.tusenergy.com/wordpress/wp-content/uploads/2020/03/%E5%9B%BE%E7%89%87-5.png)

第四步：点击**确定**按钮完成安装。

1.  使用打印机

2.  打印：正常打印文档后，刷卡，屏幕上选择打印**个人文件**，即可完成打印。

3.  复印：刷卡后按正常步骤即可完成复印。

4.  扫描：刷卡后将待扫描的文件放入送纸器（**正面朝前放）**，选择屏幕上的扫描按钮，无须再次选择自己的邮箱，屏幕上自动显示本人邮箱，点击**确认**即可。

5.  操作完成后，打印机上按“**登录**”键，退出个人登录。
