第三步：配置-\>用户信息的设定-\>用户识别符选项中选择：**输入所有者名称**，用户识别符为：本人小写姓名全拼；

![C:\\Users\\ADMINI\~1\\AppData\\Local\\Temp\\2\\d7e809af-a7e0-4757-8bbe-3d3e747bbe7f.tmp](https://raw.githubusercontent.com/baoshan2021/tc/master/333bfbd228e97ba4c9c8891770beb1eb.png)

第四步：点击**确定**按钮完成安装。

1.  使用打印机

2.  打印：正常打印文档后，刷卡，屏幕上选择打印**个人文件**，即可完成打印。

3.  复印：刷卡后按正常步骤即可完成复印。

4.  扫描：刷卡后将待扫描的文件放入送纸器（**正面朝前放）**，选择屏幕上的扫描按钮，无须再次选择自己的邮箱，屏幕上自动显示本人邮箱，点击**确认**即可。

5.  操作完成后，打印机上按“**登录**”键，退出个人登录。
